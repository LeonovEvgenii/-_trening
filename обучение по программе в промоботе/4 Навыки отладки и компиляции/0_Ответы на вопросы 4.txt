1) Как скомпилировать программу с++ с помощью gcc

Установить компилатор. Обычно с помощью apt.
Написать программу. Текстовый файл с расширением cpp.
Передать компилятору путь к написанному файлу.
Пример команды:
g++ file_name.cpp -o name_bin_execute_file
Сгенерируется файл, псоле чего запустить его
./name_bin_execute_file


2) Флаги компиляции

Для g++ примеры

-Wall - отображать предупреждения
-std=c++20 - пример компиляции под определенный стандарт
-pedantic - жесткое следование стандарту
-02 - уровень опитимизации 2 (вместо 2 может быть другое число)
-o - указание имени исполняемого файла
-S - только копиляция, без создания исходного файла


3) Как использовать отладчик gdb

Установить gdb.
Скомпилировать программу с необходимы флагом. (-g) 
Запустить отладчик с передачей скомпилированного файла.
Используя внутренний язык комманд произвести отладку.
Примеры команд:
break - устновка точки останова на определенной строке.
print - получение зпнчения переменной.
set - задание значения переменной.
run - запустить выполнение программы.


4) Точки останова

Функционал в IDE для постановки выполнения программы на паузу с отображением выполняемой строки.
В момент остановки возможно просмотр состояния переменных.
Обычно строки помечаются точками слева от номера строк.


5) Интерпретация ошибок

Из сообщений об ошибках можно либо явно понять что следует изменить для восстановления работоспособности программы,
либо косвенно догадаться, с чем может быть связано данное сообщение и с какой лругой ошибкой.

Так же сообщения можно разделить на предупреждения и ошибки.
Предупреждения желательно исправлять, но с ними запуск возможен.
