1) Что такое стек и какие операции с ним связаны?

определение - структура данных, подчиняющаяся правилу: последним вошел, первым вышел.
В с++ подключается и используется так

#include <stack>
std::stack<...>

Доступные функци:
получить последний добавленный элемент
удалить последний добавленный элемент
добавить элемент

Одна из иллюстраций применения - реализация операций отмены.
Еще иллюстрация - вычисление арифметических выражений со скобками.


2) Объясните, как работает очередь и где она применяется.

Очередь работает применяя принцип первый вошел, первый вышел.

Полдключается и используется

#include <queue>
std::queue<...>

Возможно получить первый и последний элементы.
Удалить возможно только последний вошедший.

Возможные места применения:
Обмен данными.
Обработка прерываний.
Обслуживание клиентов.


3) Как реализуется бинарное дерево поиска и какие операции с ним можно выполнять?

Реализация
Узыл (обычно выполненные в виде класса), содержат две ссылки на другие объекты класса узел.
Каждый узел хранит данные. Левые узлы хранят данные меньшие по значению, чем правые.

Применяется определенный набор териминов именно для деревьев.

Операции:
поиск (самая распространенная)
добавление
удаление
перебор


4) Перечислите основные алгоритмы сортировки и опишите один из них.

Сортировка пузырьком
Сортировка выбором
Приамидальная сортировка
Сортировка слиянием

Описание сортировки пузырьком

Организуется цикл по элементам.
Сравнивается каждый элемент со своим соеседом.
Если необходимо, то меняются местами.
Проходы повоторяются по количеству элементов в массиве.


5) Что такое граф и какие существуют способы его представления в памяти?

Определение - математическа абстракция, состоящая из узлов и связей между узлами.

В памяти может быть представлена:
матрицой смежности
списком смежности (может хранить кратные ребра)
списком ребер (не позволяет выполнить обход графа)
